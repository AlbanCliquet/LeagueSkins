# LoL Skins

League of Legends skin assets organized by champion and skin IDs.

## Structure

```
skins/
├── {champion_id}/
│   ├── {skin_id}.png
│   ├── {skin_id}.zip
│   └── {chroma_id}/
│       ├── {chroma_id}.png
│       └── {chroma_id}.zip
```
